### Example data

Because the original UK Biobank (UKBB) data used in this project is protected, this repository includes synthetically generated **example data** to demonstrate the pipeline without requiring UKBB access.

- covariates.csv — Table of covariates (columns) for each participant (rows).

- genotypes.bed — Synthetic genotype data in PLINK binary format.

- phenotypes.csv — Table of embedding dimensions (columns) for each participant (rows).
